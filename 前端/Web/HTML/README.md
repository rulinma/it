### 学习步骤

个人建议以下步骤学习，总花费时间应该是1-2周，回头学了CSS和JavaScript后继续巩固加强。

1. 学习W3C HTML教程
2. 学习黑马程序员pink老师前端入门教程
3. Udemy排名第一的 HTML5+CSS3 课程
4. 阅读官方文档[http://www.w3.org/TR/2014/REC-html5-20141028](http://www.w3.org/TR/2014/REC-html5-20141028)和标准[https://html.spec.whatwg.org](https://html.spec.whatwg.org)
5. 其他的参考学习
