# JavaScript编码规范

## 学习指南

## 规范

* JS文件命名规范
  * 字母全部小写
  * 不要带空格
  * 用破折号（-）连接单词
  * 库文件可用逗点（.），用于体现版本或从属关系

### 推荐阅读

* [JavaScript Standard Style](https://standardjs.com/rules-zhcn.html)
* [Javascript 风格指南](https://google-styleguide.readthedocs.io/zh_CN/latest/google-javascript-styleguide/contents.html)
