## 学习指南

### 核心资料


1. [Alfred](https://www.alfredapp.com) Alfred is a productivity application for Mac OS X.
2. [iTerm2](https://www.iterm2.com) iTerm2 is a replacement for Terminal and the successor to iTerm.
3. Zsh
4. [Chrome](https://www.google.cn/chrome) 浏览器。
5. [搜狗输入法](https://pinyin.sogou.com/mac) 输入法。
6. [Foxmail](https://www.foxmail.com) 邮箱管理。
7. [EverNote](https://evernote.com/intl/zh-cn) 随时随地获取、整理、分享笔记，让灵感时刻与你同行。
8. [XMind](https://www.xmind.net) 思维导图工具。
9. [FileZilla](https://filezilla-project.org) The free FTP solution.
10. [CleanMyMac](https://cleanmymac.com) Your Mac. As good as new. Mac清理管理工具。
11. [RescueTime](https://www.rescuetime.com) A personal analytics service that shows you how you spend your time and provides tools to help you be more productive. 时间花费跟踪工具。
12. [Skype](https://web.skype.com) 是超清晰网络电话工具，电脑间通话永远免费，打国内国际固定/移动电话只需市话费。
13. [Sequel Pro](http://www.sequelpro.com) Sequel Pro is a fast, easy-to-use Mac database management application for working with MySQL databases.
14. [Parall Client](https://www.parallels.com/cn/products/ras/features/parallels-client) Parallels Client is an easy-to-install client software that provides access to your Parallels RAS installations.
15. [Dash](https://kapeli.com/dash)  快速访问API。(Dash gives your Mac instant offline access to 200+ API documentation sets.)
16. [MacDown](http://macdown.uranusjr.com) MarkDown编写工具。
17. [Sublime Text](http://www.sublimetext.com)  免费的文本编辑工具。(A sophisticated text editor for code, markup and prose.)
18. [Wunderlist: To-Do List & Tasks](https://www.wunderlist.com/zh) TODO工具。奇妙清单是完成工作的最简单方法。无论你是在安排度假、与伙伴分享购物清单，或是管理多个工作项目，奇妙清单会帮助你完成所有个人和职业待办事项。
19. [Office](https://products.office.com/zh-cn/mac/microsoft-office-for-mac) 确是 Office，专为 Mac 打造。
20. [LaTeX](https://www.latex-project.org) LaTeX is a high-quality typesetting system.
21. [VirtualBox](https://www.virtualbox.org)
22. [VMware](https://www.vmware.com)
23. [Shadowsocks](https://github.com/shadowsocks/shadowsocks)
24. [github文档]https://docs.github.com/cn
